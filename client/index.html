The Technology

State-of-the-art deep learning architecture for medical image segmentation

Encoder

MobileNetV2

Pre-trained on ImageNet, providing efficient feature extraction with inverted residuals and linear bottlenecks.

Architecture

Attention U-Net

Skip connections enhanced with attention gates to focus on lesion regions while suppressing irrelevant background features.

Loss Function

Combined Loss

Optimized with BCE, Dice, and Tversky losses to handle class imbalance and improve boundary precision.

Training

Deep Supervision

Multi-scale loss computation at different decoder levels for better gradient flow and feature learning.
